mininpu · compute-tile documentation

An ASIC-bound NPU compute tile, specified end to end

A composable NPU IP template (MXU / VPU / tile / NoC) built as an IO-attached discrete accelerator. Three documents: a locked instruction-set reference plus two work-in-progress design docs. FPGA is a test platform only; the design stays ASIC-targeting.

ISA 0.5.0 · 15 opcodes geometry N=8 VREG 32 x [8x8] fp32 on-chip SPAD 32 KiB target GPT-2

01 Document set

The compute-tile design in three parts. Doc 1 is frozen; Docs 2 and 3 are being authored.

Doc 1 ISA 0.5.0 Specification Locked

The complete instruction-set reference: 15 opcodes, 64-bit fixed word, in-order single issue, no hardware interlock, and the full compiler scheduling contract.

Doc 2 Reference Microarchitecture In progress

The datapath behind the ISA: concurrency model, per-unit AGU, MXU / VPU / VREG / ACC / SPAD, and the MXU-VPU software pipeline. Locked design plus open decisions.

Doc 3 Implementation (Files & Modules) In progress

How the design maps to RTL: the tile-sequencer fold, DMU internals, and the target file/module hierarchy. Locked decisions plus pending proposals.
