mininpu Compute-Tile Documentation

ASIC-bound, composable NPU IP template · compute-tile design · ISA 0.5.0

mininpu is an ASIC-bound, composable NPU IP template (MXU / VPU / tile / NoC) built as an IO-attached discrete accelerator. This site collects the compute-tile design docs: a locked instruction-set reference plus two work-in-progress design documents. FPGA is a test platform only; the design stays ASIC-targeting.
